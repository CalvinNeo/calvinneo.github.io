基础方位

判断南北半球，点击左下角指南针归正，此时面向北方。如果能明显见到太阳，则说明太阳在北方，我们位于南半球。反之则在北半球。如果太阳出现在头顶，或者东西两侧，则不太好分辨南北半球。
或者可以让指南针指向太阳，看对着的方向。
如果太阳在头顶，那么这个地方肯定在回归线以内。
还可以通过卫星锅的夹角来判断纬度。

简介

海洋性气候、大陆性气候、季风性气候：

  1. 大陆性气候特点是降水少
  2. 海洋性气候特点是降水多且均匀，年温差小
  3. 季风气候的特点是降水差别大，年温差大

温带大陆性气候

  1. 北美洲中东部
  2. 西伯利亚
  3. 巴塔哥尼亚高原
    位于阿根廷和智利。也就是安第斯山脉东边的一长条。
  4. 欧洲部分

温带大陆性气候的植被根据纬度不同可以是荒漠、草原和针叶林。
温带大陆性气候会被一些地方细分为温带沙漠气候和温带草原气候等。

温带海洋性气候

  1. 西欧
    例如英国、比利时。
  2. 落基山脉西的北美
    例如西雅图、温哥华。
  3. 澳大利亚东南、新西兰
  4. 智利最南端

温带海洋性气候通常出现在大陆西端。
植被类型是温带落叶阔叶林。
温带海洋性气候给人一种位于青山绿水的童话世界中的感觉。

温带季风气候

  1. 中国北方的东部地区、俄罗斯远东地区南部、日本朝鲜等北部。

温带季风气候通常出现在大陆东端。
温带季风气候的主要植被是落叶阔叶林，以及针阔叶混交林。

亚热带季风气候

  1. 中国东南沿海
  2. 美国东南沿海
  3. 澳大利亚东部沿海

温带季风气候的主要植被是常绿阔叶林。

地中海气候

地中海气候是一种亚热带气候。

  1. 地中海沿岸
  2. 黑海沿岸
  3. 加州
  4. 澳大利亚珀斯、阿德莱德
  5. 南非西南
  6. 智利中部

雨热不同期。

典型植被为硬叶常绿旱生林和灌丛。
夏季过后，树林中干燥的枯枝落叶多。

热带草原气候

  1. 非洲中部
  2. 南美巴西大部
  3. 澳大利亚东部和北部

特点是全年高温，分为明显的干季和湿季。
热带草原的植物具有旱生特性，不少植物树干粗大，树皮厚，可贮存大量水分以保证在旱季能进行生命活动。代表树种有猴面包树、纺锤树、金合欢树、波巴布树等，另外是具有许多珍稀的多肉植物种类。

热带沙漠气候

  1. 撒哈拉沙漠
  2. TODO

热带季风气候

  1. 印度、斯里兰卡
  2. 泰国北部、缅甸、越南等

植被和热带雨林的差别在于：

  1. 树木多数不再常绿，每到旱季便落叶
  2. 森林结构简化，树木的密度及分层减少

所以人们称之为季雨林。

热带季风气候需要和温带大陆性气候区分，因为两者都会落叶。但热带季风更可能是干枯，可以看 Case5。

热带雨林气候

  1. 非洲西部刚果河流域，几内亚湾（也就是西南角）北岸
    注意，非洲其实大部分是热带草原和热带沙漠
  2. 马达加斯加岛东部
  3. 印尼，马来半岛南部，马来群岛，菲律宾群岛南部
  4. 亚马逊河流域
  5. 中美洲东部
  6. 澳大利亚的东北部

植被特点：

  1. 种类最丰富，树种混杂
  2. 树林有明显的分层，一般有五到八层
    最高的乔木层高达60～80米，林下有小乔木、灌木和草本植物，加上树上缠绕的藤本植物、附生植物和寄生植物
  3. 四季常青
  4. 为争夺阳光，植物都拼命往高处长

寒带气候

分为极地苔原气候和极地冰原气候。
极地苔原气候：

  1. 北美洲北部
  2. 亚欧大陆北部
  3. 格陵兰以及北冰洋群岛
  4. 马尔维纳斯群岛、南设得兰群岛、南奥克尼群岛

极地冰原气候：

  1. 格陵兰、南极大陆和北冰洋群岛
