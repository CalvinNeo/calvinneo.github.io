容易看出，这里很多东西都是一样的，例如Rc<dyn Fn(A) -> O>和impl Fn(A) -> O都表示的是F：

  1. 返回值可以用 impl 做静态分发，当然写成 Box<dyn Fn(A) -> O> 返回也是没问题的

    impl Trait only allowed in function and inherent method return types, not in closure param
    但是可以在普通函数中用
